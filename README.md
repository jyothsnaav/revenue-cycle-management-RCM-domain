# Healthcare Revenue Cycle Management Dashboard

## Overview

This project analyzes healthcare revenue cycle performance using Power BI.

## Tech Stack

- Power BI
- DAX
- Power Query
- JSON data modeling
- Data Modeling

## Features

- Claims Overview
- Payment Analysis
- Remittance Tracking
- Financial Ageing
- Payment vs Rejection Analysis
